Lokale REST-API/neuer-endpunkt
/users
Methode Pfad Beschreibung
GET /users

Abrufen aller Einträge
GET /users?[spaltenname]=[wert]

Abrufen bestimmter Einträge mit Wert
POST /users

Hinzufügen neuer Einträge mit JSON-Body: 

{
    name: "John",
    nachname: Doe",
    alter: "22",
    job: "Cashier",
}

DELETE /users?[spaltenname]=[wert]

Löschen bestimmter Einträge mit Wert
